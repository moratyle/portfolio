Brief overview

Getting Started

After viewing a simple youtube tutorial on how to connect to an API, I decided to take the project quite a few steps forward. The tutorial by the Coding Train demonstrated how to succesfully link a javascript file to read data from an API that supplied data about the ISS. He also used this data to link to a map library to give real time data of where the ISS was located.

Background

While reviewing the tutorial I decided to make my the web app more powerful and robust. This meant a few more additional features and a user interface that was intuitve and modern. I decided to display data about longitude and latitude coordinates, velocity, visibility (wheather or not the Earth's shadow was cast upon it), the country is was hovering, the number of people and their names aboard the ISS and a brief description of what the ISS is.

I felt that this was sufficent to clearly diffierentae my project from the tutorial and help myself stand out compard to other live ISS trackers.

Process

The original UI that I started with provided by the coding train

Design

This was the taking off point for the UI of the web app. The major changes were ading an automatic dark mode depending on wheather or not the ISS was covered by the sun. In addition, I wanted to go for NASA-like color theme and used some of their colors in my app.

Development (coding)

Other than displaying the current coordinates, there were no interactions. I decided to make dropdowns for the stats & astronauts as well as include a hover state when the user scrolls past the "ISS" in the title.

Final

Final Thoughts

This was a really great tutorial I took and made it into something entirely new. I would have liked to have a conversion button feature to switch bettwen miles and kilometers but I just could not get that to work properly. Also, I spent some time playing around with dropdows for a mobile view, but had trouble removing them for the desktop/ larger iPad screen sizes.

However, with those considerations, this has been my most ambitous project and I really enojoy what I built!
