Getting Started

I decided to make the classical calculator web app. Althoug there is an abundance of calculator apps, this project was a means to become more familar with CSS grids and javascript

Background

Many calculator apps are utilitarian (as they should), but why not have a dynamic calculator interface that changes on different states (basic & scientific)? This was the basis for making my app stand out from the majority.

Design

My thinking behind the UI was to make it as simple and intuitive to use. This means that the numerical and operational buttons would remain in their historic places so as not to confuse the user. To make this more custom and modern, I wanted to implement a dynamic gradient that would change based on wheather or not the user was on the basic or scientific mode.
